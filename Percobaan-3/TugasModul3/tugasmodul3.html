Tentang Saya

Nama saya Muh. Fadhlur Rahman, seorang mahasiswa Teknik Informatika di Universitas Tadulako.

Sejak kecil, saya selalu tertarik dengan komputer dan bagaimana teknologi bisa menyederhanakan kehidupan. Setelah menyelesaikan pendidikan menengah, saya memutuskan untuk melanjutkan studi di Universitas Tadulako, yang saya anggap sebagai langkah penting untuk mencapai impian saya di bidang teknologi informasi.

Di universitas, saya mempelajari berbagai topik terkait teknologi, mulai dari pemrograman dasar hingga pengembangan sistem basis data. Namun, saya merasa bahwa pemrograman web merupakan bidang yang sangat menarik dan menjanjikan di masa depan.
